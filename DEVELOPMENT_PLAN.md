# Mechanical Restaurant Horror - Development Plan

## Project Overview
- **Name:** Mechanical Restaurant Horror
- **Type:** Roblox Horror/Survival Game
- **Team:** You (coding) + Student (design)
- **Goal:** Survive 64 days by serving food, identifying weird customers, and defending against mechanical animals at night

---

## Milestones

### Phase 1: Foundation (Week 1-2)
- [ ] Set up Roblox Studio project
- [ ] Create lobby system (teleport, player count, friends-only)
- [ ] Basic day/night cycle system
- [ ] Test environment (basic restaurant interior)

### Phase 2: Core Gameplay (Week 3-4)
- [ ] Food making system (recipes, cooking UI)
- [ ] Customer spawning & ordering system
- [ ] "Weird customer" identification mechanic
- [ ] Kill button & consequences system

### Phase 3: Night Mechanics (Week 5-6)
- [ ] Mechanical animal spawning
- [ ] Window locking mechanic
- [ ] Jumpscare system
- [ ] Challenge system (food disposal)

### Phase 4: Progression (Week 7-8)
- [ ] 64-day survival system
- [ ] Gem currency & upgrades (speed, weapons)
- [ ] Game over / victory screens

### Phase 5: Polish (Week 9+)
- [ ] Sound effects & music
- [ ] Visual polish (lighting, particles)
- [ ] UI/UX improvements
- [ ] Bug testing & fixes

---

## Task Breakdown

### Scripts Needed
1. **GameManager** — Main loop, day/night cycle, game state
2. **LobbyManager** — Player joining, settings, teleporting
3. **CustomerManager** — Spawning, ordering, weird customer logic
4. **FoodSystem** — Cooking, serving, recipes
5. **CombatSystem** — Kill button, weapon upgrades
6. **NightManager** — Mechanical animal AI, spawning, attacks
7. **WindowSystem** — Lock/unlock mechanics
8. **ChallengeManager** — Food disposal task
9. **ProgressionSystem** — Day tracking, gems, upgrades
10. **UIManager** — All GUIs (HUD, menus, popups)

### Assets Needed
- Restaurant interior (tables, chairs, counter, kitchen)
- Customer NPCs (normal + weird variants)
- Mechanical animal models
- UI elements (buttons, icons, progress bars)
- Sounds (ambience, jumpscare, effects)
- Lighting (mood for day/night)
